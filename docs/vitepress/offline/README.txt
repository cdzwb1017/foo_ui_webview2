foo_ui_webview2 Offline Documentation
=====================================

Open the documentation
----------------------
1. Extract the complete ZIP to a normal folder.
2. Double-click open-docs.cmd.
3. Keep the console window open while reading.
4. Close the console window, or press Ctrl+C in it, when finished.

No installation is required. The launcher uses only components included with
Windows 10 and Windows 11: Windows PowerShell 5.1, .NET Framework networking,
and the default web browser. It does not require Node.js, Python, npm,
administrator access, or an Internet connection.

Security and privacy
--------------------
The documentation server is read-only and listens only on 127.0.0.1 (this
computer). It chooses an unused temporary port and does not accept connections
from other computers.

Troubleshooting
---------------
- Do not run open-docs.cmd from inside the ZIP preview. Extract all files first.
- If Windows shows a security prompt for files downloaded from the Internet,
  choose Run only if the ZIP came from the official foo_ui_webview2 release.
- If the browser does not open, copy the local http://127.0.0.1 address shown in
  the console into a browser.
- If files are reported missing, extract the ZIP again without changing its
  directory structure.
